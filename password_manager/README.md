# 本地密码管理器

一个基于Python开发的本地密码管理程序，具有简洁的GUI界面，能够自动生成强密码并安全地加密存储密码。

## 功能特性

### 🔐 密码生成
- 支持自定义密码长度（8-50位）
- 可选择包含大写字母、小写字母、数字、特殊字符
- 一键复制生成的密码到剪贴板

### 💾 安全存储
- 使用AES-256加密算法加密存储密码
- 主密码保护，确保数据安全
- 本地存储，不上传任何数据到服务器

### 🔍 智能检索
- 支持按账户描述和用户名搜索密码
- 实时显示所有存储的密码
- 快速复制选中的密码

### 🎯 用户界面
- 简洁直观的标签页设计
- 三个主要功能模块：生成密码、存储密码、查询密码
- 响应式布局，操作便捷

## 项目结构

```
password_manager/
├── code/                    # 程序代码目录
│   └── password_manager.py  # 主程序文件
├── env/                     # Python虚拟环境目录
└── output/                  # 输出目录
    └── passwords.json       # 加密存储的密码数据文件
```

## 依赖库

程序运行需要以下Python库：

### 必需库
- **tkinter** - Python标准GUI库（通常已内置）
- **pycryptodome** - 加密库，提供AES加密功能
- **pyperclip** - 剪贴板操作库，用于复制密码

### 安装依赖

在项目根目录下，使用以下命令安装依赖：

```bash
# 激活虚拟环境
password_manager\env\Scripts\Activate.ps1

# 安装依赖库
pip install pycryptodome pyperclip
```

或者直接使用虚拟环境中的pip安装：

```bash
password_manager\env\Scripts\pip.exe install pycryptodome pyperclip
```

## 使用方法

### 启动程序

```bash
# 使用虚拟环境中的Python运行程序
password_manager\env\Scripts\python.exe password_manager\code\password_manager.py
```

### 首次使用
1. 程序启动后会提示设置主密码
2. 输入一个安全的主密码（用于加密存储所有密码）
3. 主密码将用于后续所有加密操作

### 生成密码
1. 在"生成密码"标签页设置密码参数
2. 点击"生成密码"按钮
3. 生成的密码会显示在下方，可点击"复制到剪贴板"使用

### 存储密码
1. 在"存储密码"标签页输入账户信息
   - 账户描述（必填）：如"微信账号"、"邮箱密码"等
   - 用户名（可选）：对应的用户名或邮箱
   - 密码（必填）：可手动输入或使用生成的密码
2. 点击"存储密码"按钮保存

### 查询密码
1. 在"查询密码"标签页查看所有存储的密码
2. 在搜索框输入关键词可快速筛选
3. 选中密码后点击"复制选中密码"按钮使用

## 安全说明

- 🔒 **本地加密**：所有密码使用AES-256加密后存储在本地
- 🔑 **主密码保护**：只有输入正确的主密码才能访问存储的密码
- 📱 **离线使用**：程序完全离线运行，不连接互联网
- 🗑️ **安全删除**：删除程序时，密码数据文件也会被删除

## 技术实现

### 加密算法
- 使用PBKDF2从主密码生成32字节加密密钥
- 采用AES-256-CBC模式进行加密
- 每个密码使用随机IV（初始化向量）确保安全性

### 数据存储
- 密码数据以JSON格式存储
- 所有敏感信息都经过加密
- 数据文件位于`output/passwords.json`

### GUI框架
- 基于Python内置的Tkinter库
- 使用ttk组件实现现代化界面
- 标签页布局，功能分区清晰

## 注意事项

1. **备份主密码**：请务必记住主密码，忘记后将无法恢复存储的密码
2. **定期备份**：建议定期备份`output/passwords.json`文件
3. **安全环境**：请在可信赖的设备上使用本程序
4. **更新维护**：定期检查程序更新，确保安全性

## 故障排除

### 常见问题

**Q: 程序启动时报错"ModuleNotFoundError"**
A: 请确保已安装所有依赖库，使用虚拟环境中的pip重新安装

**Q: 忘记主密码怎么办？**
A: 主密码无法恢复，需要删除`output/passwords.json`文件重新开始

**Q: 密码无法复制到剪贴板**
A: 检查pyperclip库是否正确安装，或手动复制密码框中的内容

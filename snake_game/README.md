# 贪吃蛇游戏 (Snake Game)

一个基于Python和Pygame开发的经典贪吃蛇游戏，具有现代化的界面和功能。

## 游戏功能

- 🐍 经典贪吃蛇玩法
- 🎯 智能食物生成（避免与蛇身重叠）
- 🏆 实时最高分记录
- ⏸️ 暂停/继续功能
- 🔄 一键重新开始
- 🎮 友好的开始界面

## 游戏控制

- **方向键**：控制蛇的移动方向
- **空格键**：
  - 游戏未开始时：开始游戏
  - 游戏进行中：暂停/继续游戏
- **R键**：重新开始游戏（保留最高分）
- **ESC键**：退出游戏

## 安装要求

### 必需库

- **Python 3.6+**
- **Pygame 2.0+**

### 安装步骤

1. 确保已安装Python 3.6或更高版本
2. 安装Pygame库：
   ```bash
   pip install pygame
   ```

## 运行游戏

1. 进入游戏目录：
   ```bash
   cd snake_game
   ```

2. 运行游戏：
   ```bash
   python snake_game.py
   ```

## 游戏规则

1. 控制蛇吃掉红色食物来增加长度和分数
2. 每吃到一个食物得10分
3. 避免撞到墙壁或蛇自身
4. 游戏结束后可以查看本次游戏最高分

## 技术特点

- **面向对象设计**：使用类来组织蛇和食物的逻辑
- **中文支持**：使用系统字体支持中文显示
- **碰撞检测**：精确的边界和自身碰撞检测
- **智能食物生成**：避免食物出现在蛇身上
- **帧率控制**：稳定的游戏帧率确保流畅体验

## 注意事项

### 系统要求
- 支持Windows、macOS、Linux系统
- 需要图形界面环境
- 建议屏幕分辨率不低于800x600

### 字体要求
- 游戏使用"SimHei"字体显示中文
- 如果系统没有该字体，可能需要安装中文字体包

### 性能优化
- 游戏帧率设置为10FPS，确保游戏速度适中
- 如果感觉游戏速度过快或过慢，可以调整FPS参数

## 自定义设置

可以在代码中修改以下参数来自定义游戏：

- **窗口大小**：修改`WINDOW_WIDTH`和`WINDOW_HEIGHT`
- **游戏区域**：修改`GAME_WIDTH`和`GAME_HEIGHT`
- **网格大小**：修改`GRID_SIZE`
- **游戏速度**：修改`FPS`值
- **颜色设置**：修改颜色常量定义

## 故障排除

### 常见问题

1. **无法导入pygame**
   - 确保已正确安装Pygame：`pip install pygame`
   - 检查Python版本兼容性

2. **中文显示乱码**
   - 确保系统安装了中文字体
   - 可以尝试修改代码中的字体设置

3. **游戏运行卡顿**
   - 降低FPS值可以改善性能
   - 关闭其他占用资源的程序

### 获取帮助

如果遇到问题，请检查：
- Python和Pygame版本是否兼容
- 系统是否满足最低要求
- 控制台是否有错误信息输出
